# Tools Collection

这是一个实用工具集合，包含多个独立的工具，用于提高工作效率。

## 工具列表

### 1. DeleteTool (文件批量删除工具)
- **位置**: `/DeleteTool`
- **功能**: 用于批量删除指定文件夹中特定类型的文件
- **特点**:
  - 支持通过图形界面选择文件夹
  - 支持选择多种常见文件类型
  - 可选择是否包含子文件夹
  - 提供实时删除日志
  - 支持手动输入文件扩展名
  - 界面美观，操作简单

## 环境要求
- Python 3.x
- tkinter (Python GUI库)

## 使用说明
每个工具都在其独立的目录中，包含完整的源代码和相关资源文件。请参考各工具目录下的具体说明进行使用。

## 注意事项
- 使用删除工具时请注意备份重要文件
- 建议在执行批量操作前仔细确认操作参数
